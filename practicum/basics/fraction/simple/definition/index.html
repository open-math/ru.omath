Многоликие числа

Запишите целые числа 9,20,7,09, 20, 7, 09,20,7,0 в виде обыкновенных дробей, знаменатель которых не равен 111.

Указание

Представьте эти числа как результат какого-то деления.

Решение

Достаточно каждое из данных чисел представить в виде результата деления:

9=36:4=36420=200:10=200107=49:7=4979 = 36:4 = \frac{36}{4} \\ 20 = 200:10 = \frac{200}{10} \\ 7 = 49:7 = \frac{49}{7}9=36:4=436​20=200:10=10200​7=49:7=749​

0=02=03=04=⋯0 = \frac{0}{2} = \frac{0}{3} = \frac{0}{4} = \cdots0=20​=30​=40​=⋯

Дробь как целое число

Можно ли любую дробь представить в виде целого числа? Поясните ответ на примере.

Указание

Вдумайтесь в смысл фразы «целое число». В качестве примера возьмите какую-нибудь правильную дробь.

Решение

Само название «целое число» предполагает, что мы пользуемся ими для подсчета «целых» объектов. Поэтому нет такого целого числа, которое бы обозначало часть объекта. В качестве примера можно «половину», то есть дробь 1/21/21/2.

Получается, что несмотря на то, что любое целое число можно записать как дробь, обратное утверждение неверно.

Ответ

Нет, не любую дробь можно представить как целое число. Пример: 1/21/21/2.

Наполовину пуст или полон?

Запишите три разные обыкновенные дроби, которые выражают дробь «половина».

Указание

Представьте 333 целые пиццы. Разрежьте каждую на разное количество кусков и определите половину для каждой.

Решение

Пусть у нас есть 333 целые пиццы. Первую разрежем на два здоровенных куска. Тогда половина от этой пиццы будет выражаться дробью 1/21/21/2.

Аналогично поступаем с двумя другими пиццами, но режем их на другое количество кусков.
