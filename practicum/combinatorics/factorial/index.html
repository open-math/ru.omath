Свойства факториала

Докажите следующие свойства факториала n!n!n!:

  1. Если n>1n>1n>1, то факториал — четное число.

  2. Если n>4n>4n>4, то факториал оканчивается на 0.

  3. nn≥n!≥nn^n \geq n! \geq nnn≥n!≥n

Решение

Пункт 1

Любой факториал n, где n>1n>1n>1, будет содержать в себе множитель 2. Например:

5!=1⋅2⋅3⋅4⋅55! = 1\cdot \boxed2\cdot 3\cdot 4\cdot 55!=1⋅2​⋅3⋅4⋅5

Если число в качестве множителя содержит двойку, значит это число можно поделить на два. А если число можно поделить на два, значит это число четное.

■\blacksquare■

Пункт 2

Любой факториал n, где n>4n>4n>4, будет содержать в себе множители 2 и 5. Но 2⋅5=102\cdot 5 = 102⋅5=10, поэтому любой такой факториал будет как минимум иметь один ноль в конце.

■\blacksquare■

Пункт 3

Начнем с того, что при n=1n=1n=1 все знаки неравенства обращаются в знак равно:

1!2=11=1!=11!^2 = 1^1 = 1! = 11!2=11=1!=1

Далее будем рассматривать неравенства считая, что n>1n>1n>1.

n!≥nn! \geq nn!≥n

Слева находится произведение чисел, начиная с самого n. Это точно больше, чем одинокое n справа. Исключение — при n=2n=2n=2 получается равенство.

■\blacksquare■

nn≥n!n^n \geq n!nn≥n!

Распишем выражения слева и справа в виде цепочек умножений:

n⋅n⋅…⋅n≥1⋅2⋅…⋅nn\cdot n \cdot \ldots \cdot n \geq 1\cdot 2\cdot \ldots \cdot nn⋅n⋅…⋅n≥1⋅2⋅…⋅n

Количество множителей слева и справа одинаковое. Но слева сами числа больше, чем числа справа, поэтому и произведение слева будет больше произведения справа.

■\blacksquare■

Факторионы

Нестандартная

Факторионами называют такие числа, которые равны сумме факториалов своих цифр:

145=1!+4!+5!145 = 1! + 4! + 5!145=1!+4!+5!

Бесконечно ли количество факторионов?

Указание

Сравните минимальное n-значное число с максимальным числом, которое может получиться от суммы n факторионов.

Решение

Минимальное число из 3 цифр это 100. Из четрых цифр — 1000:

100⏟3 знака=1021000⏟4 знака=103\underbrace{100}_{3 \ знака} = 10^2 \qquad \underbrace{1000}_{4 \ знака} = 10^33 знака100​​=1024 знака1000​​=103

Замечаем, что минимальное число из n цифр это всегда 10n−110^{n-1}10n−1.

С другой стороны, максимальное число, которое может выдать сумма из n факториалов, это n раз сложить 9!:

9!+9!+…⏟n раз=n⋅9!\underbrace{9! + 9! + \ldots}_{n \ раз} = n\cdot 9!n раз9!+9!+…​​=n⋅9!

Если даже минимальное число из n цифр (10n−110^{n-1}10n−1) окажется больше, чем в теории может выдать максимальная сумма из n факториалов (n⋅9!n\cdot 9!n⋅9!), то факторионов для чисел из n цифр не существует:

10n−1>n⋅9!10^{n-1} > n\cdot 9!10n−1>n⋅9!

На это этапе кому-то уже может все стать понятно. Слева у нас степенная функция, которая растет быстрее линейной, поэтому наступит такой момент, когда левая часть станет большое правой. Впервые это случится при n=8n=8n=8. Впрочем, даже без этих знаний о функциях неравенство можно доказать.

Получается, не существует факторионов из 8 и более цифр. Это означает, что все существующие факторионы прячутся где-то среди чисел от 1 до 9 999 9999 \ 999 \ 9999 999 999. Найти их, кроме очевидных 1 и 2, можно при помощи прямого перебора на компьютере.
