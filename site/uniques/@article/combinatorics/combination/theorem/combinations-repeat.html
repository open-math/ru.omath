Обобщим метод перегородок из примера с пирожными.

Будем составлять комбинации из единиц (1) и разделителей (|). Единица означает, что мы взяли элемент, а тип этого элемента определяется положением разделителей. Пример:

111∣∣1∣11111||1|11111∣∣1∣11

Данная комбинация означает, что мы взяли 3 элемента первого типа, 0 элементов второго типа, 1 элемент третьего типа и 2 элемента четвертого типа.

Количество единиц равно количеству желаемых элементов, которые мы хотим взять, то есть k. Количество разделителей равно n−1n-1n−1, потому что они порождают как раз n групп, в которые можно вписывать единицы:

Тип 1∣Разд. 1Тип 2∣Разд. 2…∣Разд. n−1Тип nТип \ 1 \underset{Разд. \ 1}{|} Тип \ 2 \underset{Разд. \ 2}{|} \ldots \underset{Разд. \ n-1}{|} Тип \ nТип 1Разд. 1∣​Тип 2Разд. 2∣​…Разд. n−1∣​Тип n

Каждую расстановку n−1n-1n−1 разделителей типов и k единиц можно рассматривать как перестановку с повторениями. Найдем количество таких перестановок по формуле:

Cˉnk=P(n−1,k)=(n−1+k)!(n−1)! k!=…\bar{C}_n^k = P(n-1, k) = \frac{(n-1+k)!}{(n-1)! \ k!} = \ldotsCˉnk​=P(n−1,k)=(n−1)! k!(n−1+k)!​=…

Теперь в скобках в знаменателе дроби добавим и сразу вычтем k. Результат не поменяется, зато теперь выражение имеет вид формулы сочетаний из n−1+kn-1+kn−1+k по k:

…=(n−1+k)!(n−1+k−k) k!=Cn−1+kk\ldots = \frac{(\blue{n-1+k})!}{(\blue{n-1 +k}-k) \ k!} = C_{n-1+k}^k…=(n−1+k−k) k!(n−1+k)!​=Cn−1+kk​
