Красивая

Сколькими способами число n можно разложить на m слагаемых при условии, что ноль тоже считается за отдельное слагаемое?
Порядок слагаемых имеет значение!

Указание

Адаптируйте решение задачи «Анализ уравнений» или используйте метод перегородок.

Решение

Распишем число n в виде n единиц. Каждой единице назначаем свое слагаемое:

n=1Сл1, 1Сл4, 1Слm, 1Слm, …n = \overset{\small Сл_1}{1},\ \overset{\small Сл_4}{1},\ \overset{\small Сл_m}{1},\ \overset{\small Сл_m}{1},\ \ldotsn=1Сл1​, 1Сл4​, 1Слm​, 1Слm​, …

Если какому-то слагаемому или слагаемым не досталась единица, то считаем, что они равны нулю.

Каждое распределение слагаемых по единицам можно рассматривать как сочетание с повторениями из m слагаемых по n единицам. Найдем количество таких сочетаний по формуле:

Cˉmn=Cn+m−1n=(n+m−1)!n! (m−1)!\bar{C}_m^n = C_{n+m-1}^n = \frac{(n+m-1)!}{n! \ (m-1)!}Cˉmn​=Cn+m−1n​=n! (m−1)!(n+m−1)!​

Через перегородки

Задачу можно решить и методом перегородок. Распишем число n в виде n единиц:

n=1111…⏟nn = \underbrace{1111\ldots}_{n}n=n1111…​​

Между этими единицами или по бокам от них можно расставить m−1m-1m−1 «разделителей» слагаемых — перегородок (|). Если по какую-то сторону от перегородки нет единицы, то будем считать, что там стоит ноль. Примеры:

11∣1∣=2+1+0∣∣11∣∣1=0+0+2+0+1∣∣∣111=0+0+0+311|1| = 2 + 1 + 0 \\ ||11||1 = 0 + 0 + 2 + 0 + 1 \\ |||111 = 0 + 0 + 0 + 311∣1∣=2+1+0∣∣11∣∣1=0+0+2+0+1∣∣∣111=0+0+0+3

Тогда количество способов число n разложить m слагаемых, включая нули, равно количеству перестановок с повторениями из n единиц и m−1m-1m−1 перегородок:

P(n, m−1)=(n+m−1)!n! (m−1)!P(n, \ m-1) = \frac{(n + m - 1)!}{n! \ (m-1)!}P(n, m−1)=n! (m−1)!(n+m−1)!​
Ответ
Cˉmn=Cn+m−1n=(n+m−1)!n! (m−1)!\bar{C}_m^n = C_{n+m-1}^n = \frac{(n+m-1)!}{n! \ (m-1)!}Cˉmn​=Cn+m−1n​=n! (m−1)!(n+m−1)!​
Примечание

Эта задача является более конкретным вариантом замечательной задачи о композициях числа.
