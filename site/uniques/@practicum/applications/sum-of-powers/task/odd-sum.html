Сумма нечетных чисел

Найдите прямую формулу для суммы первых n нечетных чисел:

1+3+5+7+…⏟n= ?\underbrace{1 + 3 + 5 + 7 + \ldots}_{\small n} = \ ?n1+3+5+7+…​​= ?
Указание

Адаптируйте решение задачи про сумму четных чисел.

Решение

Хитрый способ

Выпишем сумму первых 2n2n2n натуральных чисел:

1+2+3+4+…+n+…+2n1 + 2 + 3 + 4 + \ldots + n + \ldots + 2n1+2+3+4+…+n+…+2n

В них ровно пополам четных и нечетных чисел. Значит, если мы вычтем из этой суммы сумму n четных чисел, то останется только сумма n нечетных чисел!

1+2+3+4+…+n+…+2n−2+4+…+2n \begin{array}{cl} &1 + \cancel{2} + 3 + \cancel{4} + \ldots + n + \ldots + \cancel{2n} \\ - \\ &\cancel{2} + \cancel{4} + \ldots + \cancel{2n} \end{array}−​1+2​+3+4​+…+n+…+2n2​+4​+…+2n​

Пользуемся формулой суммы 2n2n2n чисел и уже найденной ранее формулой суммы n четных чисел:

1+3+5+…⏟n=2n(2n+1)2−n(n+1)==n(2n+1)−n(n+1)=n(2n+1−n−1)=n2\underbrace{1 + 3 + 5 + \ldots}_{\small n} = \frac{2n(2n + 1)}{2} - n(n+1) = \\ = n(2n+1) - n(n+1) = n(2n + 1 - n - 1) = n^2n1+3+5+…​​=22n(2n+1)​−n(n+1)==n(2n+1)−n(n+1)=n(2n+1−n−1)=n2

Итак, сумма первых n нечетных чисел в точности равна квадрату этого n:

1+3+5+7+…⏟n=n2\underbrace{1 + 3 + 5 + 7 + \ldots}_{\small n} = n^2n1+3+5+7+…​​=n2

Прямое суммирование

Можно пойти напрямую, ипользуя подход из задачи про сумму четных чисел.

Нечетное число по определению это число, которое не делится на два, то есть может быть представлено в виде 2k−12k-12k−1. k в данном случае является номером нечетного числа:

1=2⋅1−13=2⋅2−15=2⋅3−1…1 = 2\cdot 1 - 1 \\ 3 = 2\cdot 2 - 1 \\ 5 = 2\cdot 3 - 1 \\ \ldots1=2⋅1−13=2⋅2−15=2⋅3−1…

Складываем все эти неравенства для всех k от 1 до n:

1+3+5+…=2⋅1−1+2⋅2−1+2⋅3−1+…+2⋅n−1=…1 + 3 + 5 + \ldots = 2\cdot 1 - 1 + 2\cdot 2 - 1 + 2\cdot 3 - 1 + \ldots + 2\cdot n - 1 = \ldots1+3+5+…=2⋅1−1+2⋅2−1+2⋅3−1+…+2⋅n−1=…

Двойку можно вынести за скобки. Всего вычитается n единиц:

…=2(1+2+3+…+n)−n=n(n+1)−n=n(n+1−1)=n2\ldots = 2(1+2+3+\ldots + n) - n = n(n+1) - n = n(n+1-1) = n^2…=2(1+2+3+…+n)−n=n(n+1)−n=n(n+1−1)=n2

Итак, сумма первых n нечетных чисел в точности равна квадрату этого n:

1+3+5+7+…⏟n=n2\underbrace{1 + 3 + 5 + 7 + \ldots}_{\small n} = n^2n1+3+5+7+…​​=n2
Ответ
1+3+5+7+…⏟n=n2\underbrace{1 + 3 + 5 + 7 + \ldots}_{\small n} = n^2n1+3+5+7+…​​=n2
Примечание

У этой задачи есть красивое геометрическое доказательство. Из нечетных чисел можно делать угловые рамки, которые при совмещении образуют квадрат:

1+3+5+7+9⏟5=52\underbrace{1 + 3 + 5 + 7 + 9}_{\small 5} = 5^251+3+5+7+9​​=52
