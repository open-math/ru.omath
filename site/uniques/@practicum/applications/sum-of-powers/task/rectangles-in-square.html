Прямоугольники в квадрате

Красивая

Сколько прямоугольников можно построить на квадратном поле n на n из квадратных клеток?

Указание

Выберите узел поля и посчитайте, сколько можно построить прямоугольников, для которых этот узел будет являться левой верхней вершиной.

Придумайте удобный алгортим перебора узлов.

Решение

Начнем с левого верхнего угла поля. Всего в верхней строчке можно построить n прямоугольников высотой в 1 клетку, n прямоугольников высотой в 2 клетки, ..., n прямоугольников высотой в n клеток. Получается, из левого верхнего узла поля можно построить n⋅nn\cdot nn⋅n прямоугольников:

Смещаемся на узел вниз и строим прямоугольники уже из левого второго сверху узла. По горизонтали каждый раз будет получатся все так же n прямоугольников, но вот их высота уже будет ограничена — максимум n−1n-1n−1. Поэтому всего n⋅(n−1)n\cdot (n-1)n⋅(n−1) прямоугольников можно построить из левого второго сверху узла:

Продолжая смещать узел строго по одной клетке вниз, мы каждый раз будем получать все меньшее количество прямоугольников: n⋅(n−2)n\cdot(n-2)n⋅(n−2), n⋅(n−3)n\cdot (n-3)n⋅(n−3) и так далее, потому что будет уменьшаться максимально допустимая их высота.

Найдем общее количество прямоугольников из узлов на «первой вертикали» поля:

n⋅n+n⋅(n−1)+n⋅(n−2)+…+n⋅1n\cdot n + n\cdot (n-1) + n\cdot (n-2) + \ldots + n\cdot 1n⋅n+n⋅(n−1)+n⋅(n−2)+…+n⋅1

Выносим за скобку n:

n(n+(n−1)+(n−2)+…+1)n(n + (n-1) + (n-2) + \ldots + 1)n(n+(n−1)+(n−2)+…+1)

В скобках получили сумму первых n натуральных чисел. Пока что прямую формулу вставлять не будем. Вместо нее используем введенное обозначение:

nSn1nS_n^1nSn1​

Теперь поднимаемся обратно наверх и в этот раз смещаемся направо на одну клетку. Так мы попадаем во вторую «вертикаль» поля. И повторяем точно такой же процесс: строим все прямоугольники из второго слева верхнего узла, потом снижаемся на клетку вниз, снова строим, потом еще ниже и так до самого низа вертикали:

В этот раз все прямоугольники будут ограничены в длину — максимум (n−1)(n-1)(n−1). Всего прямоугольников на второй вертикали поля:

(n−1)⋅n+(n−1)⋅(n−1)+…+(n−1)⋅1(n-1)\cdot n + (n-1)\cdot(n-1) + \ldots + (n-1)\cdot 1(n−1)⋅n+(n−1)⋅(n−1)+…+(n−1)⋅1

Выносим за скобки (n−1)(n-1)(n−1) и сумму первых n чисел заменяем обозначением:

(n−1)Sn1(n-1)S_n^1(n−1)Sn1​

Повторям этот процесс для оставшихся вертикалей поля. Весь алгоритм заключается в движении по узлам вертикали поля до упора вниз, потом скачет на следующую вертикаль и снова до упора вниз:

Находим общее количество всех возможных прямоугольников, построенных по нашему алгоритму:

n⋅Sn1⏟1 вертикаль+(n−1)⋅Sn1⏟2 вертикаль+…+2⋅Sn1⏟Предпоследняя+1⋅Sn1⏟Последняя\underbrace{n\cdot S_n^1 }_{\text{1 вертикаль}} + \underbrace{(n-1)\cdot S_n^1}_{\text{2 вертикаль}} + \ldots + \underbrace{2\cdot S_n^1}_{\text{Предпоследняя}} + \underbrace{1\cdot S_n^1}_{\text{Последняя}}1 вертикальn⋅Sn1​​​+2 вертикаль(n−1)⋅Sn1​​​+…+Предпоследняя2⋅Sn1​​​+Последняя1⋅Sn1​​​

Выносим за скобки одинаковые суммы:

Sn1(n+(n−1)+…+2+1)=Sn1⋅Sn1=(Sn1)2S_n^1(n + (n-1) + \ldots + 2 + 1) = S_n^1\cdot S_n^1 = \left(S_n^1\right)^2Sn1​(n+(n−1)+…+2+1)=Sn1​⋅Sn1​=(Sn1​)2

Итак, количество всех возможных прямоугольников, которые можно построить в квадратной сетке n на n равно квадрату суммы первых n натуральных чисел, или же сумме кубов n натуральных чисел:

(1+2+…+n)2=13+23+…+n3=n2(n+1)24(1+2+\ldots + n)^2 = 1^3 + 2^3 + \ldots + n^3 = \frac{n^2(n+1)^2}{4}(1+2+…+n)2=13+23+…+n3=4n2(n+1)2​

Обратите внимание, что каждая «группа» прямоугольников уникальная — она начинается со своего узла поля. Именно поэтому среди всех построенных таким образом треугольников нет дубликатов!

Ответ

Количество всех возможных прямоугольников, которые можно построить в квадратной сетке n на n равно квадрату суммы первых n натуральных чисел, или же сумме кубов n натуральных чисел:

(1+2+…+n)2=13+23+…+n3=n2(n+1)24(1+2+\ldots + n)^2 = 1^3 + 2^3 + \ldots + n^3 = \frac{n^2(n+1)^2}{4}(1+2+…+n)2=13+23+…+n3=4n2(n+1)2​
