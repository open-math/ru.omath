–ü–æ–ª—É—á–∞–µ—Ç—Å—è, –¥–≤–∞–∂–¥—ã —Å–ª–æ–∂–∏–≤ –≤—Å–µ —á–∏—Å–ª–∞ –æ—Ç $1$ –¥–æ $100$ –º—ã $100$ —Ä–∞–∑ –ø–æ–ª—É—á–∏–ª–∏ —á–∏—Å–ª–æ $101$:

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –µ—Å—Ç—å —á–∏—Å–ª–æ $n$ –≤–æ –≤—Å–µ—Ö —Å—Ç–µ–ø–µ–Ω—è—Ö: $n^2, \ n^1$ –∏ –¥–∞–∂–µ $n^0 = 1$.
–í—ã–ø–∏—à–µ–º –ø–æ–¥–æ–±–Ω—ã–µ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏—è –∫–≤–∞–¥—Ä–∞—Ç–∞ —Ä–∞–∑–Ω–æ—Å—Ç–∏ **–¥–ª—è –≤—Å–µ—Ö** —á–∏—Å–µ–ª –æ—Ç $1$ –¥–æ $n$:

–ü–æ—Å–º–æ—Ç—Ä–∏—Ç–µ, –∫–∞–∫ —Å—Ç–æ–ª–±–∏–∫–∏ —Å–∏–Ω–∏—Ö —á–∏—Å–µ–ª –æ–±—Ä–∞–∑—É—é—Ç —Ü–µ–ø–æ—á–∫–∏ —á–∏—Å–µ–ª –≤–æ –≤—Å–µ—Ö —Å—Ç–µ–ø–µ–Ω—è—Ö.
–¢–µ–ø–µ—Ä—å —Å–ª–æ–∂–∏–º –≤—Å–µ —ç—Ç–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –¥—Ä—É–≥ —Å –¥—Ä—É–≥–æ–º (–æ—Ç–¥–µ–ª—å–Ω–æ –ª–µ–≤—ã–µ –∏ –æ—Ç–¥–µ–ª—å–Ω–æ –ø—Ä–∞–≤—ã–µ —á–∞—Å—Ç–∏), –≤—ã–Ω–æ—Å—è –∑–∞ —Å–∫–æ–±–∫–∏ $-2$:

@math

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ —Å–ª–µ–≤–∞ –∏ –≤ —Å–∫–æ–±–∫–∞—Ö —Å–ø—Ä–∞–≤–∞ —É –Ω–∞—Å –æ–±—Ä–∞–∑–æ–≤–∞–ª–∏—Å—å —Å—É–º–º—ã —á–∏—Å–µ–ª –≤ —Ä–∞–∑–Ω—ã—Ö —Å—Ç–µ–ø–µ–Ω—è—Ö!

@math
    \underbrace{1^2 + 2^2 + \ldots + (n-1)^2}_{\small S_{n-1}^2} = \\ =
    \underbrace{(1^2 + 2^2 + \ldots + n^2)}_{\small S_n^2} -2 \underbrace{(1 + 2 + \ldots + n)}_{\small S_n^1} + \underbrace{(1 + 1 + \ldots + 1)}_{\small{S_n^0}}

–ó–∞–º–µ–Ω—è–µ–º –¥–ª–∏–Ω–Ω—ã–µ —Å—É–º–º—ã –Ω–∞ [–≤–≤–µ–¥–µ–Ω–Ω–æ–µ –æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ](d:sum-of-powers) –¥–ª—è —Å—É–º–º —Å—Ç–µ–ø–µ–Ω–µ–π —á–∏—Å–µ–ª:

$$ S_{n-1}^2 = S_n^2 -2 S_n^1 + S_n^0 $$

–ö—Ä–∞—Å–∏–≤–æ–µ –ø–æ–ª—É—á–∏–ª–æ—Å—å —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ...<br>

–†–∞–Ω–µ–µ –º—ã —É–∂–µ –≤—ã—è—Å–Ω–∏–ª–∏, —á—Ç–æ $S_n^0$ [—Ä–∞–≤–Ω–æ](t:sum-of-ones) $n$.

$$ S_n^1 = \frac{1}{2}\left( S_n^2 - S_{n-1}^2 + n \right) $$

–û—Å—Ç–∞–ª–æ—Å—å —Ç–æ–ª—å–∫–æ –Ω–∞–π—Ç–∏, —á–µ–º—É —Ä–∞–≤–Ω–∞ —Ä–∞–∑–Ω–æ—Å—Ç—å $S_n^2 - S_{n-1}^2$.

–ö–∞–∂–¥—ã–π —Ä–∞–∑, –∫–æ–≥–¥–∞ —Ä–µ—á—å –∑–∞—Ö–æ–¥–∏—Ç –æ —Å—Ç–µ–ø–µ–Ω—è—Ö –∏ –¥–ª–∏–Ω–Ω—ã—Ö —Å—É–º–º–∞—Ö, —ç—Ç–æ –ø–æ–≤–æ–¥ –∑–∞–¥—É–º–∞—Ç—å—Å—è –æ [–±–∏–Ω–æ–º–µ –ù—å—é—Ç–æ–Ω–∞](article|combinatorics/binomial-theorem).
–û–Ω —á–∞—Å—Ç–æ –≤—ã—Å—Ç—É–ø–∞–µ—Ç —Å–≤–æ–µ–æ–±—Ä–∞–∑–Ω—ã–º "–º–æ—Å—Ç–∏–∫–æ–º" –º–µ–∂–¥—É —Å—É–º–º–∞–º–∏ –∏ –∏—Ö –æ–±—â–∏–º–∏ —Ñ–æ—Ä–º—É–ª–∞–º–∏.

–ï—Å–ª–∏ —É –≤–∞—Å –≤–æ–∑–Ω–∏–∫–Ω—É—Ç —Å–ª–æ–∂–Ω–æ—Å—Ç–∏, —Ç–æ —Å–Ω–∞—á–∞–ª–∞ —Å–∞–º–æ—Å—Ç–æ—è—Ç–µ–ª—å–Ω–æ –≤—ã–≤–µ–¥–∏—Ç–µ —É–∂–µ –æ–ø–∏—Å–∞–Ω–Ω—ã–µ –≤—ã—à–µ —Ñ–æ—Ä–º—É–ª—ã –¥–ª—è [–ø–µ—Ä–≤–æ–π](heading:sum-of-numbers-proof), [–≤—Ç–æ—Ä–æ–π](t:sum-of-squares) –∏ [—Ç—Ä–µ—Ç—å–µ–π](t:sum-of-cubes) —Å—Ç–µ–ø–µ–Ω–µ–π –∞–ª–≥–µ–±—Ä–∞–∏—á–µ—Å–∫–∏–º —Å–ø–æ—Å–æ–±–æ–º.

–†–∞—Å—Å–º–æ—Ç—Ä–∏–º —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–µ —Å–ª–µ–¥—É—é—â–µ–π —Å—Ç–µ–ø–µ–Ω–∏ –±–∏–Ω–æ–º–∞:

{ .freeze }
$$ (n-1)^{k+1} = C_{k+1}^0 \ \blue{n^{k+1}} - C_{k+1}^1 \ \blue{n^{k}} + C_{k+1}^2 \ \blue{n^{k-1}} - \ldots + (-1)^{k+1}C_{k+1}^{k+1} \ \blue{n^{0}} $$

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –≤ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏–∏ –≤—Å—Ç—Ä–µ—á–∞—é—Ç—Å—è –≤—Å–µ —Å—Ç–µ–ø–µ–Ω–∏ —á–∏—Å–ª–∞ $n$ –æ—Ç $k$-–æ–π, –¥–æ –Ω—É–ª–µ–≤–æ–π.
–í—ã–ø–∏—à–µ–º –≤—Å–µ –≤–æ–∑–º–æ–∂–Ω—ã–µ —Ä–∞–∑–ª–æ–∂–µ–Ω–∏—è —á–∏—Å–µ–ª –æ—Ç $0$ –¥–æ $n$:

{ .freeze }
@math
    (1-1)^{k+1} = C_{k+1}^0 \ \blue{1^{k+1}} - C_{k+1}^1 \ \blue{1^{k}} + C_{k+1}^2 \ \blue{1^{k-1}} - \ldots + (-1)^{k+1}C_{k+1}^{k+1} \ \blue{1^{0}}
    \\
    (2-1)^{k+1} = C_{k+1}^0 \ \blue{2^{k+1}} - C_{k+1}^1 \ \blue{2^{k}} + C_{k+1}^2 \ \blue{2^{k-1}} - \ldots + (-1)^{k+1}C_{k+1}^{k+1} \ \blue{2^{0}}
    \\
    (3-1)^{k+1} = C_{k+1}^0 \ \blue{3^{k+1}} - C_{k+1}^1 \ \blue{3^{k}} + C_{k+1}^2 \ \blue{3^{k-1}} - \ldots + (-1)^{k+1}C_{k+1}^{k+1} \ \blue{3^{0}}
    \\
    \cdots
    \\
    (n-1)^{k+1} = C_{k+1}^0 \ \blue{n^{k+1}} - C_{k+1}^1 \ \blue{n^{k}} + C_{k+1}^2 \ \blue{n^{k-1}} - \ldots + (-1)^{k+1}C_{k+1}^{k+1} \ \blue{n^{0}}

–°–ª–æ–∂–∏–º –≤—Å–µ —ç—Ç–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –¥—Ä—É–≥ —Å –¥—Ä—É–≥–æ–º (–æ—Ç–¥–µ–ª—å–Ω–æ –ª–µ–≤—ã–µ –∏ –æ—Ç–¥–µ–ª—å–Ω–æ –ø—Ä–∞–≤—ã–µ —á–∞—Å—Ç–∏).
–í –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –≤—ã–Ω–æ—Å–∏–º –∑–∞ —Å–∫–æ–±–∫–∏ –æ–¥–∏–Ω–∞–∫–æ–≤—ã–µ [–±–∏–Ω–æ–º–∏–∞–ª—å–Ω—ã–µ –∫–æ—ç—Ñ—Ñ–∏—Ü–∏–µ–Ω—Ç—ã](article|combinatorics/binomial-theorem|heading:binomial-coeff):

@math
    0^{k+1} + 1^{k+1} + 2^{k+1} + \ldots + (n-1)^{k+1} = \\ =
    C_{k+1}^0(1^{k+1} + \ldots + n^{k+1}) - C_{k+1}^1(1^{k} + \ldots + n^{k}) + \ldots + (-1)^{k+1}C_{k+1}^{k+1}(1^0 + \ldots + n^0)

–¢–µ–ø–µ—Ä—å –∑–∞–º–µ–Ω–∏–º —Å—É–º–º—É –≤ –ª–µ–≤–æ–π —á–∞—Å—Ç–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –∏ —Å—É–º–º—ã –≤ —Å–∫–æ–±–∫–∞—Ö –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ –Ω–∞ –≤–≤–µ–¥–µ–Ω–Ω–æ–µ [–æ–±–æ–∑–Ω–∞—á–µ–Ω–∏–µ](d:sum-of-powers) —Å—É–º–º —Å—Ç–µ–ø–µ–Ω–µ–π —á–∏—Å–µ–ª:

$$ S_{n-1}^{k+1} = C_{k+1}^0 S_n^{k+1} - C_{k+1}^1 S_n^{k} + \ldots + (-1)^{k+1}C_{k+1}^{k+1}S_n^0 $$

@important
    main: |
        <ab-strong>–ù–µ–≤–µ—Ä–æ—è—Ç–Ω–æ!</ab-strong>
        –í—Å–µ–≥–æ –ø–∞—Ä–æ–π –ø—Ä–æ—Å—Ç—ã—Ö –¥–µ–π—Å—Ç–≤–∏–π —Å –±–∏–Ω–æ–º–∞–º–∏ –ù—å—é—Ç–æ–Ω–∞ –º—ã –ø–æ–ª—É—á–∏–ª–∏ –µ–¥–∏–Ω—É—é —Ñ–æ—Ä–º—É–ª—É, –≤ –∫–æ—Ç–æ—Ä—É—é "–≤–ø–µ—á–∞—Ç–∞–Ω—ã" –≤—Å–µ —Å—É–º–º—ã $n$ —á–∏—Å–µ–ª –≤—Å–µ—Ö —Å—Ç–µ–ø–µ–Ω–µ–π (–æ—Ç $0$ –¥–æ $k+1$)!

–ù–æ –Ω–∞–º –≤—Å–µ –∂–µ –Ω—É–∂–Ω–∞ —Å—É–º–º–∞ $n$ —á–∏—Å–µ–ª –≤ $k$-–π —Å—Ç–µ–ø–µ–Ω–∏, –∞ –Ω–µ —Å—É–º–º–∞ $n$ —á–∏—Å–µ–ª $k+1$-–π —Å—Ç–µ–ø–µ–Ω–∏.
–ü–æ—ç—Ç–æ–º—É –µ—â–µ –Ω–µ–º–Ω–æ–≥–æ –ø—Ä–µ–æ–±—Ä–∞–∑—É–µ–º —Ä–∞–≤–µ–Ω—Å—Ç–≤–æ.

–î–ª—è –Ω–∞—á–∞–ª–∞ –∏–∑–±–∞–≤–∏–º—Å—è –æ—Ç —Å—É–º–º $k+1$-–π —Å—Ç–µ–ø–µ–Ω–∏, –∏—Å–ø–æ–ª—å–∑—É—è —Ç–æ—Ç —Ñ–∞–∫—Ç, —á—Ç–æ —Ä–∞–∑–Ω–æ—Å—Ç—å $S_n^{k+1} - S_{n-1}^{k+1}$ [—Ä–∞–≤–Ω–∞](spoiler:diff-general-explain) $n^{k+1}$:

{ #diff-general-explain }
@spoiler

{ .freeze }
$$ 0 = \underbrace{\underbrace{C_{k+1}^0}_{\small 1} S_n^{k+1} - S_{n-1}^{k+1}}_{\small n^{k+1}} -C_{k+1}^1 S_n^k + \ldots + (-1)^{k+1}C_{k+1}^{k+1}S_n^0 $$

{ .freeze }
$$ 0 = n^{k+1} -C_{k+1}^1 S_n^k + \ldots + (-1)^{k+1}C_{k+1}^{k+1}S_n^0 $$

{ .freeze }
$$ S_n^k = \frac{1}{\underbrace{C^1_{k+1}}_{k+1}} \left( n^{k+1} + C^2_{k+1}S^{k-1}_n - \ldots + (-1)^{k+1}C_{k+1}^{k+1}S_n^0 \right) $$

–ù–∞–∫–æ–Ω–µ—Ü, –≤ –ø—Ä–∞–≤–æ–π —á–∞—Å—Ç–∏ —Ä–∞–≤–µ–Ω—Å—Ç–≤–∞ –∏—Å–ø–æ–ª—å–∑—É–µ–º —Å–∏–º–≤–æ–ª —Å—É–º–º—ã –∏ –ø–æ–ª—É—á–∞–µ–º –æ–∫–æ–Ω—á–∞—Ç–µ–ª—å–Ω—É—é —Ñ–æ—Ä–º—É–ª—É:

## –ü—Ä—è–º–∞—è —Ñ–æ—Ä–º—É–ª–∞?

–ü–æ–ª—É—á–µ–Ω–Ω–æ–π –Ω–∞–º–∏ —Ä–µ–∫—É—Ä—Ä–µ–Ω—Ç–Ω–æ–π [—Ñ–æ—Ä–º—É–ª—ã](t:recurrent) –≤–ø–æ–ª–Ω–µ –¥–æ—Å—Ç–∞—Ç–æ—á–Ω–æ –¥–ª—è –≤—ã–≤–æ–¥–∞ –ø—Ä—è–º—ã—Ö —Ñ–æ—Ä–º—É–ª —Å—É–º–º —á–∏—Å–µ–ª –Ω–µ–±–æ–ª—å—à–∏—Ö —Å—Ç–µ–ø–µ–Ω–µ–π.
